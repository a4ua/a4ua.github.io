Golang学习之路—基本数据类型

Go基础知识
Golang
发布日期:   2020-10-13
更新日期:   2020-10-13
文章字数:   2.6k
阅读次数:  

一、变量（Variable）

  • 变量使用前必须要声明，同一作用域内不能重复声明，声明后的非全局变量必须要被使用，否则编译报错。

  • 所有在 Go中的创建的变量都必须要进行初始化。若不指定，则Go会使用类型对应的0值进行初始化。而保证Go会使用0值进行初始化的是var变量。因此声明变量要使用var。

    • int <===> 0
    • float <===> 0.0
    • bool <===> false
    • string <===> “”（空字符串）
    • 切片、函数、指针 <===> nil 
    func main() {
    var a int
    var b string
    var c float64
    var d bool
    var e *int

    fmt.Printf("var a int  %T [%v]\n", a, a)     //var a int  int [0]
    fmt.Printf("var b string  %T [%v]\n", b, b)  //var b string  string []
    fmt.Printf("var c float64  %T [%v]\n", c, c) //var c float64  float64 [0]
    fmt.Printf("var d bool  %T [%v]\n", d, d)    //var d bool  bool [false]
    fmt.Printf("var e *int  %T [%v]\n", e, e)    //var e *int  *int [<nil>]
}
0x00 变量声明及初始化

var 变量名 变量类型

  • 变量声明

    //标准声明
var name  string
var age   int
var isOk  bool

//批量声明
var (
    name string
    age int
    isOk bool
)

  • 变量初始化

    //声明后初始化
var name string
name = "Ein"

//声明同时初始化
var age int = 18

//Go可以根据值自动进行类型推导,因此在声明并初始化时可省略类型。
var age = 18

//批量声明初始化
var name,age = "Ein",18

  • 短变量

    //':'可以当作var，只能在函数中使用，函数外部全局变量不可用，因为函数外部的语句都必须以关键字开头。
// 定义变量同时要显式初始化
// 变量在之前必须是未声明的。
name := "Ein"
age := 18
name,age := "Mikasa",18

    • 附加知识：在多个短变量声明和赋值中，至少有一个新声明的变量出现在左值中，即便其他变量名可能是重复声明的，编译器也不会报错。

      conn,err := net.Dial("tcp","localhost:80")
conn1,err:= net.Dial("tcp","localhost:80")  //err重复声明，但由于有新变量conn1声明，所以不会报错

  • 匿名变量

    // '_'下划线可以用来作为匿名变量，他不会占空间。
// 通常用于接收无用的函数返回值
x, _ := func()
0x01 变量的作用域
  • 函数内定义的变量称为局部变量
  • 函数外定义的变量称为全局变量
    • 全局变量必须使用var关键字定义。
    • 要想让外部包在import后可使用，变量名的首字母必须大写。

Go语言中的变量、函数、常量名称的首字母也可以大写，如果首字母大写，则表示它可以被外部其它的包访问。

二、常量（Constant）

  • 常量即一旦定义，在程序运行期间不能被改变值。

  • 常量只存在于编译时，即使定义在函数内部也是如此，并且只能是布尔型、数字型（整数型、浮点型和复数）和字符串型。

  • 常量的值可以是表达式，但是必须是能够在编译时就能够确定的。

  • 常量定义

    const pi = 3.14

//b、c的值会以a的值作为值。
const (
    a = 10
    b
    c
)

  • iota：iota第一次出现时，会被置为0。之后每增加一行，iota的值会增加1。

    const (
    n1 = iota       //0
    n2              //1
    n3              //2
)

const (
    n1 = iota       //0
    _
    n3              //2
)

const (
    n1 = iota       //0
    n2 = 10 
    n3              //2
)

const (
    n1, n2 = iota + 1, iota + 2   //1,2。两个iota都为0
    _, _
    n3, n4             //3,4。两个iota为2。
)

const (
    Ldate         = 1 << iota //  1 :  1 左移 0位.  0000 0001
    Ltime                     //  2 :  1 左移 1位.  0000 0010
    Lmicroseconds             //  4 :  1 左移 2位.  0000 0100
    Llongfile                 //  8 :  1 左移 3位.  0000 1000
    Lshortfile                // 16 :  1 左移 4位.  0001 0000
    LUTC                      // 32 :  1 左移 5位.  0010 0000
)

三、基本数据类型

0x00 整型

  • Go语言同时提供了有符号和无符号类型的整数运算。

    • int8、int16、int32 和 int64 为有符号整数类型，分别对应 8、16、32、64bit 大小的有符号整数。
    • uint8、uint16、uint32 和 uint64 对应无符号整数类型。

  • int 和 uint的大小是根据对应特定 CPU 平台的字长而定的，在32位系统为4个字节长（int32）， 在64位系统为8个字节长(int64)。

  • byte类型实际为uint8。

  • rune 类型实际是 int32 ，通常用于Unicode 字符，表示一个 Unicode 码点(字符在字符集中的唯一编号)。

  • uintptr，没有指定具体的 bit 大小但是足以容纳指针，常用于底层编程。

  • 有符号整数使用最高 bit 位用来表示符号位，因此

    • 一个 n bit 的有符号数的取值范围是从 -2^(n-1) 到 2^(n-1)-1
    • 无符号取值范围： 0 到 2^(n-1)
0x01 浮点型
  • Go语言提供了两种精度的浮点数，float32 和 float64。
0x02 布尔类型
  • 一个布尔类型的值只有两种：true 和 false。
0x03 字符

字符使用单引号''。

func main() {
    // 下面四个是等价的
    var a byte = 'A'
    var b byte = 65
    var c uint8 = '\x41'
    var d byte = '\101'

    // 下面三个等价，rune用于宽字符
    var e rune = '波'
    var f int32 = '波'
    var g int = '\u6CE2' //用到 4 字节，则使用\u前缀，如果需要使用到 8 个字节，则使用\U前缀。

    fmt.Printf("%c、%c、%c、%c、%c、%c、%c", a, b, c, d, e, f, g)  //A、A、A、A、波、波、波
}
0x04 字符串

  • 字符串是一个不可改变的字节序列，字符串用双引号""。

  • 字符串数据结构

    • 字长（word）即一个字的长度，字长由系统平台位数决定，与操作系统地址长度相同，32位系统字长为4字节，64位系统字长度8字节。

    • 字符串的本质是数组（array）。

    • 一个字符串的数据结构使用双字（double word）存储。

      • 第一个字（word）存储的是一个指针，指针指向字符串第一个字符。

      • 第二个字（word）存储的是字符串的长度。

        //Go使用0值进行初始化时的字符串的数据结构。第一个字（指针）为nil，第二个字（长度）为0。
-------
| nil |
-------
|  0  |
-------

  • Go 支持2种形式的字符串字面值：

    • 解释字符串：使用双引号括起来的字符序列，不能跨行书写，其中的相关的转义字符将被替换。

    • 非解释字符串：使用反引号“`”括起来，支持换行，反引号包含的内容会被原原本本的作为字符。

      str1 := "abc\n"   //\n会被解释为换行
str2 := `abc\n`   //\n会原原本本的输出

  • 字符串操作

    • 字符串长度（所占字节长度）：len(str)

    • 字符串可索引得到字符str[n]，但是只适用于纯ascii字符的字符串，因为索引的是某字节值，而像中文字符由多个字节保存，就会索引不出来。

    • 字符串拼接

      func main() {
    str1 := "this is"
    str2 := "Mikasa"

    //可以使用“+”拼接字符串，+=也可以。
    str3 := str1 + " " + str2

    str4 := fmt.Sprintln(str1, "", str2)
    str5 := fmt.Sprintf("%s %s \n", str1, str2)

    fmt.Println(str3) //this is Mikasa
    fmt.Println(str4) //this is  Mikasa
    fmt.Println(str5) //this is Mikasa
}

    • 字符串遍历

      func main() {
    str := "Hello,阿库娅"

    //这种方式是以byte类型来处理，到中文时会出现乱码，即会把字符串转为[]byte切片。。
    for i := 0; i < len(str); i++ {
        fmt.Printf("%T\n", str[i]) //uint8
        fmt.Printf("%v(%c)、", str[i], str[i])
    }
                 
    //这种方式是以rune类型来处理，可以正常打印出字符串，即会把字符串转为[]rune切片。
    for _, i := range str {
        fmt.Printf("%T\n", i) //int32
        fmt.Printf("%v(%c)、", i, i)
    }
}

    • 字符串修改

      //字符串是不能被修改的（只读），要想修改字符串，必须要先将字符串转换为[]byte或[]rune类型的切片，本质上是进行了复制。

//转换，修改
func main() {
    str := "Hello,阿库娅"
    new_str := []rune(str)

    new_str[2] = 'L'
    new_str[3] = 'L'
    new_str[8] = '雅'

    fmt.Printf("%s", string(new_str))
}

    • 字符串函数

      func main() {
    str := "abcdecfg" 

    ret := strings.Split(str, "c") //分割，[ab de fg]
    strings.Join(ret, "+")         //合并，ab+de+fg

    strings.Contains(str, "cd")   //是否包含字串，true
    strings.HasPrefix(str, "abc") //是否以字串为前缀，true
    strings.HasSuffix(str, "cfg") //是否以字串为后缀，true
    strings.Index(str, "cd")      //字串第一次出现的索引，2
}

四、字符串格式化占位符

func main() {
    var a string = "zzzz"
    var b int = 1

    fmt.Printf("%T %T\n", a, b) //变量的类型
    fmt.Printf("%v %v\n", a, b) //变量的值

    fmt.Printf("%d\n", b)       //十进制形式整形
    fmt.Printf("%b\n", b)       //二进制形式整形
    fmt.Printf("%o\n", b)       //八进制形式整形
    fmt.Printf("%x\n", b)       //十六进制形式整形

    fmt.Printf("%s\n", a)       //字符串
    fmt.Printf("%c\n", a[0])    //字符
}

五、类型转换

  • Go语言只有强制类型转换，没有隐式类型转换。

  • 只有支持相互转换的类型之间才能进行强制类型转换。

    • 字符串与切片
    • 整形与浮点型

T(表达式) //将括号中的转化为类型T。

var num int = 100
f := float32(num)

var str string = "abcde"
slc := []byte(str)
slc1 := []rune(str)

六、指针

  • 使用指针只有一个目的，就是为了共享内存。

  • 指针变量存储的是指针指向的变量的地址。

    • 在变量名前面添加&操作符（前缀）来获取变量的内存地址（取地址操作）。

    • 对指针变量进行取值操作使用*操作符，可以获得指针变量指向的原变量的值。

      • *操作符作为右值时，意义是取指针的值，
      • *作为左值时，也就是放在赋值操作符的左边时，表示指针指向的变量。

  • Go语言的类型指针不能进行偏移和运算，只能对数据进行修改。

    func swap(a, b *int) {
    t := *a
    *a = *b  //*操作符作为右值时，意义是取指针的值。作为左值时，表示指针指向的变量。
    *b = t
}

func main() {
    var x, y = 1, 2
    fmt.Println(x, y) //1 2

    swap(&x, &y)
    fmt.Println(x, y) //2 1
}

  • new创建指针

    func main() {
    var p1 *int

    p2 := new(int)

    fmt.Println(p1) //<nil>
    fmt.Println(p2) //0xc000064058,某个int类型变量地址
}

func main() {
    ptr1 := new(string)
    ptr2 := new(int)
    ptr3 := new([]byte)
    fmt.Printf("%T、%T、%T\n", ptr1, ptr2, ptr3)    //*string、*int、*[]uint8
    fmt.Printf("%v、%v、%v\n", *ptr1, *ptr2, *ptr3) //""、0、[]

    *ptr1 = "Hello"
    *ptr2 = 2
    *ptr3 = []byte{1, 2, 3}
    fmt.Printf("%v、%v、%v\n", *ptr1, *ptr2, *ptr3) //Hello、2、[1 2 3]
}
    • new与make
      • 二者均用于分配内存空间。
      • new用于基本数据类型（int、string、float、struct）,函数返回对应类型的指针。
      • make用于slice、map、chan,函数返回类型本身。
