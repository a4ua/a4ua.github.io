Golang学习之路—结构体

Go基础知识
Golang

一、自定义类型与类型别名

  • 类型别名在编译时会被替换为原类型。

  • 自定义类型在编译时会保留。

    func main() {
    type newInt int  //自定义类型，新的类型
    type myInt = int //类型别名

    var m newInt = 100
    var n myInt = 100

    fmt.Printf("%T\n", m) //main.newInt
    fmt.Printf("%T\n", n) //int
}

二、结构体（Struct）

  • Go语言的结构体与其他语言的类有同样地位，但是Go的结构体没有继承等面向对象特性。
  • 结构体是一块连续的地址空间，封装一个或多个基本数据类型，实现复合类型。
  • 结构体是一种自定义数据类型，特别：每个结构体都是一种自定义的类型。
  • 结构体是值类型。
0x00 结构体定义与实例化

  • 结构体的定义只是内存布局的描述，只有当实例化结构体或初始化结构体时，才真正分配内存。

    //语法格式
type 结构体名 struct {
    字段名1 类型
    字段名2 类型
    字段名3 类型
}

//例子
type Person struct {
    name  string
    age   int
    hobby []string
}

  • 实例化结构体

    • 其一：使用var关键字，会自动用0值进行初始化。

      var aqua Person
fmt.Printf("%v\n", aqua)   //{  0 []}

    • 其二：使用new函数，返回结构体指针。

      aqua := new(Person)

fmt.Printf("%v\n", aqua)    //&{ 0 []}
fmt.Printf("%v\n", *aqua)   //{ 0 []}

    • 其三：在 Go语言中，对结构体进行&取地址操作时，视为对该类型进行一次 new 的实例化操作。

      aqua := &Person{}

fmt.Printf("%v\n", aqua)  //&{ 0 []}
fmt.Printf("%v\n", *aqua) //{ 0 []}

    • 结构体构造函数：约定成俗以 new+结构体名 为构造函数名。

      type Person struct {
    name string
    age  int
}

// 尽量返回指针而不是结构体本身以减少拷贝（结构体是值类型）节约内存资源。
func newPerson(name string, age int) *Person {
    return &Person{
        name: name,
        age:  age,
    }
}

func main() {
    p1 := newPerson("Mikasa", 17)
    p2 := newPerson("Aqua", 1000)

    fmt.Println(p1) //&{Mikasa 17}
    fmt.Println(p2) //&{Aqua 1000}
}

  • 结构体属性初始化

    //第一种
var p1 Person
p1.Name = "Yuigahama"
p1.Age = 17
fmt.Println(p1, p1.Name, p1.Age) //{Yuigahama 17} Yuigahama 17

//第二种：声明并初始化
p2 := Person{
    Name: "Mikasa",
    Age:  16,
}
fmt.Println(p2, p2.Name, p2.Age) //{Mikasa 16} Mikasa 16

//第三种：new
p3 := new(Person)
p3.Name = "Tosaki"
p3.Age = 18
fmt.Println(p3, p3.Name, p3.Age) //&{Tosaki 18} Tosaki 18

// 第四种
p4 := &Person{
    "Toka",
    17,
}
fmt.Println(p4, p4.Name, p4.Age) //&{Toka 17} Toka 17
0x01 匿名结构体

  • 匿名结构体常用于结构体只需要单实例时。

    func main() {
    waife := struct {
        name string
        age  int
    }{
        name: "Tosaki",
        age:  17,
    }

    fmt.Printf("%v\n", waife) //{Tosaki 17}
}
    func main() {
    var waife struct {
        name string
        age  int
    }
    waife.name = "Yuigahama"
    waife.age = 17

    fmt.Printf("%v\n", waife) //{Yuigahama 17}
}
0x02 结构体是值类型

  • 结构体是值类型，因此作为函数参数传入时，传入的是拷贝。

  • 结构体指针访问成员变量时可以直接使用 .运算符 。这是因为 Go语言为了方便开发者访问结构体指针的成员变量，使用了语法糖（Syntactic sugar）技术，自动将 p.name 形式转换为 (*p).name。

    type Person struct {
    name, gender string
    age          int
    flag         bool
}

func change1(p Person) {
    p.flag = true
}

func change2(p *Person) {
    // (*p).flag = true
    p.flag = true  //go自动实现语法糖（*p）
}

func main() {
    var p1 Person

    p1.flag = false
    fmt.Printf("%v\n", p1) //{    0 false}
                
    change1(p1)   
    fmt.Printf("%v\n", p1.flag) //false

    change2(&p1)
    fmt.Printf("%v\n", p1.flag) //true
}
0x03 结构体方法与接收器

  • Go方法是作用在接收器的一个函数，而接收器是某种类型。

  • 接收器可以是几乎任意类型（接收器不能是接口类型），即几乎任意类型都可拥有方法。

  • 方法的作用对象是接收器，而函数没有作用对象。

  • 为接收器定义一个方法可理解为其它语言中为类定义方法，GO语言的接收器即某一类型。

  • 为类型定义方法

    • 接收器名类似于python的self，php的$this，谁调用了方法就指向哪个实例。但官方建议使用接收器类型名的第一个小写字母作为接收器名，而不是self、this。

      func (接收器名 接收器类型) 方法名(参数列表) (返回参数) {

      ​ 函数体

      }

  • 值接收器与指针接收器

    • 值接收器的接收器名变量是调用方法实例的拷贝。

    • 指针接收器的接收器名变量是指向调用方法实例的指针。

      type Person struct {
    name string
    age  int
}

// 值接收者，p是调用该方法的实例的拷贝
func (p Person) birthday1() {
    p.age++
}

// 指针接收者，p是指向调用该方法的实例的指针。
func (p *Person) birthday2() {
    p.age++
}

func main() {
    p1 := Person{
        name: "Mikasa",
        age:  17,
    }

    p1.birthday1()
    fmt.Println(p1.age) //17。值接收器修改不生效，因为修改的是拷贝。

    p1.birthday2()
    fmt.Println(p1.age) //18。指针接收器修改生效。
}

    • 总结

      • 方法要修改结构体成员内容，使用指针接收者。
      • 方法不涉及修改结构体成员内容，使用值接收者。

  • Go语言可以使用type定义出自定义类型，之后我们可以为新自定义的类型定义方法。

    当我们想让某种类型增加某个功能，就可以使用type生成新的类型，然后对新的类型添加方法，就可以达到既拥有原有类型所有特性，又可以添加实现某些附加的功能。

    package main
import "fmt"

type New_int int

func (n New_int) Is_Zero() bool {
        if n == 0 {
            return true
        } else {
            return false
        }
}

func main() {
    var test New_int = 0
    fmt.Print(test.Is_Zero())   //true
}
0x04 匿名字段结构体

  • 使用匿名结构体要求同一种类型只能出现一次。

    func main() {
    type Test struct {
        string
        int
    }

    t := Test{
        "Aqua",
        14,
    }

    fmt.Println(t.string) //Aqua
    fmt.Println(t.int)    //14
}
    func main() {
    type address struct {
        province string
        city     string
    }

    type Person struct {
        name    string
        age     int
        address //匿名字段
    }

    p1 := &Person{
        name: "Aqua",
        age:  1000,
        address: address{
             province: "GZ",
             city:     "DY",
        },
    }

    fmt.Println(p1) //&{Aqua 1000 {GZ DY}}

    fmt.Println(p1.address.province) //GZ
    fmt.Println(p1.province)         //GZ。与上面等价，可以直接取，但注意同名冲突。
}

  • 模拟实现继承

    type animal struct {
    name  string
    color string
}

func (a *animal) move() {
    fmt.Printf("%s move move move !!!", a.name)
}

type Cat struct {
    name   string
    animal //匿名结构体
}

func main() {
    c := &Cat{
        name:   "feipei",
        animal: animal{name: "feipei", color: "yellow"},
    }

    fmt.Println(c) //&{feipei {feipei yellow}}
    c.move()       //feipei move move move !!!。继承了animal结构体的move方法。
                   //其实是c.animal.move()
}
