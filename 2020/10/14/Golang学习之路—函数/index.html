Golang学习之路—函数

Go基础知识
Golang
发布日期:   2020-10-14
更新日期:   2020-10-14
文章字数:   1.3k
阅读时长:   5 分
阅读次数:  

一、函数定义

  • Go是编译型语言，因此函数的编写顺序是无关紧要的，通常把main函数放于前面提高可读性。

  • 普通函数声明：fun关键字、函数名、形参、返回值列表（可省略）、函数体

    func 函数名（arg1,arg2…）（返回值列表）{ … return … }

    func add(x int, y int) string {  //int类型的形参x,y,返回值类型为string类型。
    return x + y
}

func sub(x, y int) (z int) { //形参类型相同可简写，明确指定z变量为返回值
        z = x - y   //z变量在函数体也不用再声明了
        return      //自动找z变量返回
}

func sum(x, y int, a, b string) (int, string) {
        return 0, "a"
}

func first(x int, _ int) int { return x } 

func zero(int, int) int { return 0 }

  • 可变长参数

    func add(x ...int) {     //可变长参数必须是最后一个形参
        fmt.Println(x)   //[1 2 3 4]。x是一个切片，可变参数必须是最后一个参数。
}

func main() { 
        add(1, 2, 3, 4)
}

  • 一些总结：

    • 如果函数不需要返回值，那么可以省略返回值列表，若写了返回值列表，则必须使用return返回值。
    • 返回值可以是变量名+类型，也可以直接是类型。
    • 形参和返回值的变量名对于调用者而言是没有意义的，Go语言不存在类型于python一样的key=value传参方式。
    • Go语言形参不存在默认值的概念。
    • 函数不能嵌套函数。（函数内只能用匿名函数）

二、函数类型

  • 函数是有类型的，函数的签名就是其类型。即如果两个函数的形参类型及返回值类型对应相同，那么这两个函数具有相同的类型。

  • 函数类型可以作为形参的类型

    func calc(deal func(int, int) int, y, z int) int {
        return deal(y, z)
}

func sum(a, b int) int {
        return a + b
}

func sub(a, b int) int {
        return a - b
}

func main() {
        fmt.Println(calc(sum, 1, 2)) //3
        fmt.Println(calc(sub, 3, 1)) //2
}

  • 函数类型也可以作为返回值类型。

三、函数参数

  • Go语言的参数传递，值类型默认都是值传递，即传递的是拷贝。若要传递引用，可以使用指针，即在变量名前加&。

  • 切片（slice）、字典（map）、接口（interface）、通道（channel）这样的引用类型都是默认使用引用传递（即使没有显式的指出指针）。

  • 可变长参数

    func test1(nums []int) {
        fmt.Println(nums) //[1 2 3]
}

func test2(nums ...int) {
        fmt.Printf("%T\n", nums) //[]int
        fmt.Println(nums)        //[4 5 6]
}

func main() {
        test1([]int{1, 2, 3})
        test2(4, 5, 6)
}
    • 可变参数本质上是使用切片来接收，但是Go使用了语法糖技术，即 ...int 等同于 []int ，在函数内部二者没差异，差异体现在调用的时候，可以看到没有实现语法糖的函数调用需要传入切片。
    • 要获得传入的多个参数就通过遍历切片。
    • 要实现接收任意类型的多个参数可以使用： ...interface{}

四、匿名函数

  • 匿名函数即没有函数名的函数，声明方式于普通函数声明方式的不同之处也仅在于没有函数名。

  • 匿名函数作回调函数：

    func handle(list []string, callback func(string)) {

        for _, v := range list {
                callback(v)
        }
}

func main() {
        list := []string{"Mikasa", "Toka", "Tosaki", "Yuigahama", "Mashiro"}
        handle(list, func(name string) { fmt.Printf("%s 赛高！！\n", name) })
}

五、闭包

  • 闭包即一个能记住嵌套作用域变量值的函数，尽管那个作用域或许已经不存在。
  • 闭包 = 函数 + 上下文环境

六、defer

  • defer关键字定义的语句在函数返回(return)之前才会被执行，若定义了多个defer语句，则逆序执行，后定义的先执行。

  • defer通常用于释放连接（socket连接、数据库连接、文件句柄等）

  • return语句不是原子操作，而是分为两个步骤：返回值赋值—>RET。而defer处于两者之间执行。

    //代码示例1
func f1() int {
        x := 5

        defer func() {
            x++    //2.修改了x的值，而不是返回值
        }()

        return x  //1.将x的值赋值给返回值，由于没有指明返回值，因此GO内部会使用一个内存空间存储，x并不是返回值，而是将x的值给返回值。
}

//代码示例2
func f2() (x int) {

        defer func() {
                x++  //2.修改了x，x是返回值，即修改了返回值
        }()

        return 5 //1.将5赋值给返回值x
}

//代码示例3
func f3() (y int) {

        x := 5
        defer func() {
                x++ //2.修改了x的值，返回值y的值没有被修改。
        }()

        return x //1.将x的值5赋值给返回值y
}

//代码示例4
func f4() (x int) {

        defer func(x int) {
                x++ //2.修改了x的值，但该x是外层函数x的拷贝。
        }(x)

        return 5 //1.将5赋值给返回值x
}

func main() {
        fmt.Println(f1()) //5
        fmt.Println(f2()) //6
        fmt.Println(f3()) //5
        fmt.Println(f4()) //5
}

七、内置函数

  • close() 主要用来关闭channel。
  • len() 用于获取长度，比如string、array、slice、map、channel。
  • cap() 用于获取容量。
  • copy() 用于拷贝。
  • new 用来分配内存，主要用来分配值类型，比如int、struct。返回的是指针。
  • make() 用来分配内存，主要用来分配引用类型，比如chan、map、slice。
  • append() 用来追加元素到slice中。
  • panic和recover用来做错误处理。
