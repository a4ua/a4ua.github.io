Go Package—flag

Go Package
Golang
发布日期:   2020-10-29
更新日期:   2020-10-29
文章字数:   1k
阅读时长:   4 分
阅读次数:  

flag包是Go内置的用于解析程序命令行参数的包，本质上是对os.Args的封装。

os.Args

os.Args是一个切片，里面存储着命令行参数，第一个元素是程序名，后面的元素是命令行参数。

// test.go
func main() {
    fmt.Println(os.Args)
}
  • 运行程序：test.exe --name aqua -a=17
    • 打印结果：[test.exe --name aqua -a=17]

flag基础用法

使用flag包解析命令行参数通常包括两个步骤，第一个步骤是定义用于接收命令行参数的flag，第二步是调用flag.Parse()函数进行解析，该函数会将命令行参数（os.Args）解析到在第一步中定义的flag中。

简单例子：

func main() {
    //第一步：定义接收命令行参数的flag
    mode := flag.String("mode", "", "scan mode")   //定义字符串类型的参数
    th := flag.Int("thread", 4, "num of thread")   //定义Int类型的参数

    //第二步：解析命令行参数到上一步定义的flag中。
    flag.Parse()

    //go run test.go --mode fast --thread 14
    fmt.Println(*mode)  //fast
    fmt.Println(*th)    //14
}

Step 1：Define flags

flag包支持的命令行参数类型有bool、int、int64、uint、uint64、float float64、string、duration。

如何理解命令行参数类型呢？

实际上从os.Args获取到的参数都是字符串(string)，而flag包提供的对应类型接收参数的函数，会帮我们做string到Go内置类型的转换。

func main() {

    age := flag.Int("age", 3, "help message")
    checked := flag.Bool("check", false, "help message")

    flag.Parse()

    //go run test.go --age 18 --check true
    fmt.Printf("%T\n", *age)     //int
    fmt.Printf("%T\n", *checked) //bool
    fmt.Printf("%T\n", os.Args)  //[]string
}

在定义接收命令参数的flag时，每种类型的命令行参数接收都有两种定义方法。

  • flag.Type()和flag.TypeVar()

    • Type是具体的类型(int、string…)。

      func main() {

    age1 := flag.Int("age1", 3, "help message")

    var age2 int
    flag.IntVar(&age2, "age2", 3, "help message")

    flag.Parse()

    //go run test.go --age1 18 --age2 19
    fmt.Printf("%T、%v\n", age1, *age1) //*int、18
    fmt.Printf("%T、%v\n", age2, age2)  //int、19
}

      • flag.Int()具有返回值，返回指针。flag.IntVar()无返回值，但需要传入接收参数的变量的指针，使用时直接是值。

      • 同理的还有：flag.String()与flag.StringVar()、flag.Bool()与flag.BoolVar()等等其它类型。

flag函数参数详解

以字符串(string)类型举例，其它类型同理

  • func String(name string, value string, usage string) *string

    • name：要接收的命令行参数的名称。
    • value：默认值。
    • usage：帮助信息。
    • *string：命令行参数的值的指针。

  • func StringVar(p *string, name string, value string, usage string)

    • p：用于接收命令行参数的值的指针。
    • name：要接收的命令行参数的名称。
    • value：默认值。
    • usage：帮助信息。

命令行参数格式

支持的flag格式如下：

  • -flag / --flag
  • -flag=x / --flag=x
  • -flag x / --flag x （不能用于布尔类型）

几点总结：

  • -与--是等价的，可以随意。
  • 布尔类型参数不能用上面第三种格式指定参数值。
  • 布尔类型参数的值可以用：1, 0, t, f, T, F, true, false, TRUE, FALSE, True, False。
  • 时间间隔类型(time.Duration)的参数值可以用：如300ms、1.5h、2h45m。
    • 合法的单位有ns、us /µs、ms、s、m、h。

其它函数

// NArg is the number of arguments remaining after flags have been processed.
func NArg() int { return len(CommandLine.args) }

// Args returns the non-flag command-line arguments.即返回位置参数。
func Args() []string { return CommandLine.args }

// Arg returns the i'th command-line argument.
func Arg(i int) string { return CommandLine.Arg(i) }

// NFlag returns the number of command-line flags that have been set.
func NFlag() int { return len(CommandLine.actual) }
