Hexo + GitPage博客搭建

Hexo
教程
发布日期:   2020-09-20
更新日期:   2020-09-21
文章字数:   545
阅读次数:  

前备环境

  • Git
  • Node.js

安装Hexo

npm install -g hexo-cli

初始化Blog

hexo init Myblog #会在当前目录下生成Myblog目录，即博客目录。

安装依赖模块

cd Myblog

npm install #根据Myblog目录下的package.json安装所需的模块。如hexo-server等。

目录、文件结构说明：

Myblog

​ I— node_modules # 存放模块的目录。

​ |— scaffolds # 文章、页面模板。

​ |— source # 存放文章的目录。

​ |— themes # 存放Hexo主题的目录。

​ |— .gitignore

​ |— _config.yml #Hexo主配置文件，如网站Title、主题等配置。

​ |— package.json #依赖说明文件。

​ |__ package-lock.json

安装主题

主题在Hexo主配置文件 _config.yml中的theme配置项中设置。

cd Myblog/themes/

git clone https://github.com/blinkfox/hexo-theme-matery.git

额外插件

a.代码高亮

npm i -S hexo-prism-plugin

修改Hexo主配置文件 _config.yml：

highlight:     #关闭hexo自带的代码高亮
  enable: false

prism_plugin:    #新增这部分配置以启用prism高亮插件。
  mode: 'preprocess'    # realtime/preprocess
  theme: 'tomorrow'
  line_number: false    # default false
  custom_css:

b.字数统计

npm i --save hexo-wordcount

在主题的配置文件 _config.yml中启用字数统计：

postInfo:
  wordCount: true      # 文章字数统计
  totalCount: true      # 站点总文章字数

c.搜索插件

npm install hexo-generator-search --save

在Hexo的主配置文件 _config.yml中新增以下内容：

search:
  path: search.xml
  field: post

新增分类、关于我等页面

具体页面配置，见主题说明。

hexo new page “categories”

hexo new page “tags”

hexo new page “about”

部署到GitPages

安装将博客到部署到Git的插件

npm install hexo-deployer-git –save

配置Git信息

git config –global user.email “you@example.com“
git config –global user.name “a4ua”

在Hexo的主配置文件中配置：

deploy:
  type: 'git'
  repo:  https://github.com/a4ua/a4ua.github.io

部署命令

hexo d

Hexo常用命令

启动本地Hexo服务器

cd Myblog

hexo s #服务器默认监听在http://localhost:4000

创建新的文章

hexo new “title” #根据scaffolds\post.md模板生成文章。

渲染

hexo g #渲染会生成一个public文件夹，里面存放渲染好的博客文件。

部署

hexo d #将博客部署到GitPage，实质是上传public文件夹到github对应仓库。

清理缓存

hexo clean
